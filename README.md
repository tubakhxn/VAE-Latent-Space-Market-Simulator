## Project Overview

This project is a Variational Autoencoder (VAE) Latent Space Market Simulator. It is designed to simulate and analyze financial market data using a VAE, a type of generative machine learning model. The simulator fetches historical asset data, processes it, and uses a VAE to model the latent space of market dynamics, enabling the study of market regimes and synthetic data generation.

## How It Works

1. **Data Fetching**: The simulator downloads historical price data for selected financial assets.
2. **Preprocessing**: The data is cleaned and prepared for modeling, including normalization and feature engineering.
3. **VAE Modeling**: A Variational Autoencoder is trained on the processed data to learn a compressed representation (latent space) of market behavior.
4. **Simulation & Analysis**: The latent space is explored to simulate new market scenarios, analyze regime changes, and generate synthetic market data.
